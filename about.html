Ferramentas Open-Source

Sou adepto do software livre e do open-source. Utilizo como sistema operacional Linux Mint, hoje o mais utilizado das versões Linux existentes até o presente momento (março 2017), muito semelhante ao Windows 7, tal Linux favorece em muito na transição e adaptação do sistema operacional GNU Linux.

Inkscape

Inkscape é um software livre para editoração eletrônica de imagens e documentos vetoriais, com base numa versão mais avançada do antigo Sodipodi no qual teve origem. Trata-se assim de um fork considerado de sucesso.

Gimp

O Gimp é um editor de imagens e desenho vetorial. O projeto foi criado em 1995 por Spencer Kimball e Peter Mattis, dois estudantes, como uma alternativa livre ao Adobe Photoshop. Foi um projeto universitário que amadureceu bastante e hoje alcança grande popularidade, sendo utilizado por muitos profissionais.

Atom

O Atom é um editor de texto de código aberto (Open Source) disponível para as plataformas MacOS, Linux e Windows, desenvolvido, em sua maioria, pela comunidade no GitHub sob a licença do MIT. Além disso, permite a instalação de pacotes estendidos, todos com licenças de Software Livre, construídos e mantidos pela comunidade.
